CSS/JavaScript Problems

Each of the problems below will require styling something or using JavaScript to make it perform some action. These are not meant to require complex solutions and in most cases, there is a fairly simple approach that can be taken. All problems, except for problem 1, must be solved without changing the provided HTML or the contents of base.css. Please comment your code where appropriate. When complete, please zip and upload all files to a file sharing site like Google Drive and share the link with your recruiter.
